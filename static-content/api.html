Insert Table

You can insert a record to a jsonbox by using HTTP POST to: /api/insert/${DBNAME}/${PASSWORD}

Example request JSON body:

{
    "title":  "Harry Potter",
    "author": "JK Rowling",
    "body":   "Once Upon a time",
    "info": { 
        "id": 198, 
        "publisher": "Ayush" 
    },
    "hidden": false,
    "meta": {
        "votes": 152,
        "price":  45.89
    }
}

Example response JSON body:

{
    "status": "OK"
}

{
    "status": "Incorrect Password"
}

{
    "status": "Could not find table"
}

{
    "status": "Something went wrong with getting database name or password."
}

Drop Table

You can insert a record to a jsonbox by using HTTP POST to: /api/drop/${DBNAME}/${PASSWORD}

Example response JSON body:

{ 
    status : "OK" 
}

{ 
    status : "Incorrect password" 
}

{ 
    status : "Could not drop table." 
}

{ 
    status : "Something went wrong with getting database name or password." 
}

Find In Table

You can insert a record to a jsonbox by using HTTP POST to: /api/find/${DBNAME}/${PASSWORD}

Example request JSON body:

{
    "title" : "Harry Potter",
    "author" : "JK Rowling"
}

{
    //empty request will return all records
}

Example response JSON body:

//Your response record. List of records.
[
    {
        "info": {
            "id": 198,
            "publisher": "Timothy"
        },
        "meta": {
            "votes": 152,
            "price": 45.89
        },
        "_id": "5f375719ac9781001e5d7c74",
        "title": "Harry Potter",
        "author": "JK Rowling",
        "body": "Once Upon a time",
        "hidden": false,
        "__v": 0
    }
]

{ 
    status : "Could not query table." 
}

Find One And Delete

You can insert a record to a jsonbox by using HTTP POST to: /api/findOneAndDelete/${DBNAME}/${PASSWORD}

Example request JSON body:

{
    "title" : "Harry Potter"
}

Example response JSON body:

//Item removed
{
    "info": {
        "id": 4324324,
        "publisher": "Ayush"
    },
    "meta": {
        "votes": 4324324234,
        "price": 432849032
    },
    "_id": "5f375719ac9781001e5d7c74",
    "title": "Harry Potter",
    "author": "JK Rowling",
    "body": "Once Upon a time",
    "hidden": false,
    "__v": 0
}

//No items removed
null

{ 
    status : "Could not find and remove from table." 
}

{ 
    status : "Incorrect password" 
}

{ 
    status : "Something went wrong with getting database name, password or parameters" 
}
